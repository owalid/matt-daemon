Welcome to Matt-Daemon

Description: Matt-Daemon is a simple daemon write in C++, You can send command or text input to the daemon and it will interpret it.

Commands available:

Tips: You can click on the command it will be automatically write in the textarea.

clear: This command is used to clear the log file.

archive: This command is used to archive logs.

procevent start: This command is used to start procevent listener mode.

procevent stop: This command is used to stop procevent listener mode.

quit: This command is used to quit the daemon.
